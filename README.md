# Intervention Management Backend

## Description

Backend d'une application de gestion des interventions terrain permettant de planifier, suivre et gérer les interventions réalisées par les techniciens.

## Technologies utilisées

- Java 17
- Spring Boot
- PostgreSQL
- Spring Data JPA
- Hibernate
- API REST
- JWT

## Fonctionnalités principales

### Authentification
- Connexion des utilisateurs
- Gestion des rôles Manager et Technicien

### Gestion des interventions
- Création des interventions
- Affectation des techniciens
- Suivi des statuts
- Consultation des détails

### Rapports et pièces jointes
- Ajout des rapports d'intervention
- Transmission des photos et fichiers associés
